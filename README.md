# Object-Oriented Programming in C#

Homework assignments and exercises from the C# OOP course, covering object-oriented programming principles and design patterns.

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 1 | Defining Classes | Classes, constructors, properties, methods |
| 2 | Static Members and Namespaces | Static fields, methods, namespaces organization |
| 3 | Other Types in OOP | Structs, enums, generic types |
| 4 | Inheritance and Abstraction | Inheritance, abstract classes, interfaces |
| 5 | Encapsulation and Polymorphism | Access modifiers, method overriding, polymorphism |
| 6 | Exception Handling | Try-catch, custom exceptions, best practices |

## How to Run

1. Open the `.sln` solution file in Visual Studio
2. Set the desired project as the startup project
3. Press `F5` to build and run

Alternatively, use the .NET CLI:

```bash
cd <homework-folder>/<project-folder>
dotnet run
```

> **Note:** These are course homeworks from Software University (SoftUni). They are intended for educational purposes and demonstrate OOP concepts in C#.
